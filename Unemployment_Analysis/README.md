# Unemployment Analysis with Python

## Objective
Analyze unemployment trends and study the impact of Covid-19.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Project Structure
Modular design with reusable components imported into main.py.

## Key Insights
- Unemployment increased significantly during Covid-19
- Gradual recovery observed post-pandemic
- Regional disparities exist in unemployment rates

## Conclusion
This analysis provides valuable insights that can support economic and policy-level decisions.
